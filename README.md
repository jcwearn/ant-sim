# Ant-Sim
###Author:
Jackson Wearn
###Description:
A simple application that provides a visualization for ant tracking data
###Install:
```
git clone git@github.com:jcwearn/ant-sim.git
cd ant-sim
npm install
```
###Usage:
```
npm start
```
Then navigate to localhost:3000 in an internet browser.
